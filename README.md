# ⚽ Footballium

Footballium — це сучасна комплексна веб-платформа для управління футбольними турнірами та командами. Проєкт об'єднує всіх учасників процесу: адміністраторів ліг, тренерів, суддів та вболівальників в одній екосистемі з преміальним дизайном.

## 🌟 Основні можливості (Ролі)

- **👑 ADMIN**: управління турнірами, новинами, користувачами
- **📋 COACH**: управління командою, заявки на турніри, склад на матч
- **哨 REFEREE**: ведення матчу в реальному часі (голи, картки, заміни)
- **👀 USER**: перегляд турнірів, статистики, коментарі, улюблені команди

## 🎨 Дизайн

Платформа використовує унікальний висококонтрастний фірмовий стиль **Black & Lime**:
- Глибокі чорні та темно-сірі фони (`#000000`, `#0a0a0a`, `#1c1a1a`).
- Яскраво-лаймові акценти (`#CCFF00`) для активних елементів та кнопок.
- Сучасна типографіка з використанням великих літер (uppercase) для заголовків.

## 🛠 Технологічний стек

- **Фреймворк**: Next.js 14+ (App Router), TypeScript
- **Стилізація**: Tailwind CSS + shadcn/ui
- **База даних**: Prisma ORM + PostgreSQL (Neon)
- **Автентифікація**: Custom JWT auth (jose + bcryptjs)
- **Реалтайм**: Pusher Channels — real-time оновлення матчів
- **Медіа**: Cloudinary — зображення (логотипи, фото гравців)
- **Редактор**: @tiptap/react — rich text editor для новин

## 🚀 Встановлення та запуск (Локально)

Щоб розгорнути платформу на своєму локальному комп'ютері, виконайте наступні кроки:

1. **Клонуйте репозиторій**:
   ```bash
   git clone https://git.ztu.edu.ua/ipz/2022-2026/ipz-22-3/biloshytskyy-maksym/footballium.git
   cd footballium
   ```

2. **Встановіть залежності**:
   ```bash
   npm install
   ```

3. **Створіть `.env` файл та додайте змінні середовища**:
   ```env
   DATABASE_URL=
   JWT_SECRET=
   PUSHER_APP_ID=
   PUSHER_KEY=
   PUSHER_SECRET=
   PUSHER_CLUSTER=
   NEXT_PUBLIC_PUSHER_KEY=
   NEXT_PUBLIC_PUSHER_CLUSTER=
   CLOUDINARY_CLOUD_NAME=
   CLOUDINARY_API_KEY=
   CLOUDINARY_API_SECRET=
   RESEND_API_KEY=
   RESEND_FROM_EMAIL=
   NEXT_PUBLIC_APP_URL=
   ```

4. **Налаштуйте базу даних**:
   ```bash
   npx prisma db push
   npx prisma db seed
   ```

5. **Запустіть сервер розробки**:
   ```bash
   npm run dev
   ```

6. **Відкрийте у браузері**: [http://localhost:3000](http://localhost:3000)

## 🧩 Реалізовані модулі

- **Auth**: реєстрація з вибором ролі, JWT, middleware захист.
- **Турніри**: CRUD, заявки команд, автогенерація сітки (Round Robin / Single Elimination).
- **Команди & Гравці**: CRUD, Cloudinary фото, пошук.
- **Live-матч**: Pusher, протокол судді, перегляд наживо для глядачів.
- **Статистика**: турнірна таблиця, бомбардири, авто-перерахунок.
- **Новини**: CRUD, коментарі, чернетка/публікація.
- **Налаштування**: профілі по ролях, керування користувачами, логи активності.
- **Email**: підтвердження реєстрації, статус заявки, призначення судді.

## 📄 Структура проєкту

- `/src/app/(public)` — Публічна частина сайту (Новини, Команди, Матчі).
- `/src/app/(coach)` — Захищена зона для тренерів (Дашборд, Налаштування складу).
- `/src/app/(admin)` — Панель адміністратора (Керування користувачами, Новини, Дашборд).
- `/src/app/(referee)` — Інтерфейс судді для ведення матчу (Live-протоколи).
- `/src/app/(settings)` — Налаштування профілю для всіх ролей користувачів.
- `/src/app/api` — API роути.
- `/src/app/auth` — Сторінки автентифікації.
- `/src/components` — Перевикористовувані UI компоненти (кнопки, картки, таблиці).
- `/prisma` — Схема бази даних та скрипт заповнення початкових даних (`seed.ts`).
